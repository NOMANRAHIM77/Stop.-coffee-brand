# STOP. // THE SYSTEM
### AN INDUSTRIAL COFFEE EXPERIENCE

**STOP.** is a high-performance, brutalist e-commerce interface built for those who treat caffeine as a technical requirement. No fluff. No rounded corners. Just pure utility and high-contrast design.

---

## 🛠 TECH STACK

- **Framework:** [React.js](https://reactjs.org/) (Vite)
- **State Management:** [Redux Toolkit](https://redux-toolkit.js.org/) (Cart & Auth Management)
- **UI Components:** [Shadcn UI](https://ui.shadcn.com/) (Radix Primitives)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Neo-Brutalism)
- **Form Handling:** [React Hook Form](https://react-hook-form.com/)
- **Validation:** [Zod](https://zod.dev/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router DOM](https://reactrouter.com/)

---

## 🚀 CORE FEATURES

### 1. Global State with Redux Toolkit
- **`authSlice`**: Manages user session state, handling login, logout, and persistent user data across the app.
- **`cartSlice`**: Handles complex shopping cart logic, including item addition/removal, quantity tracking, and real-time total calculations.

### 2. Shadcn UI Integration
- **Dialogs & Modals:** Used for high-impact login feedback and system alerts.
- **Form Components:** Leverages Shadcn's accessible primitives for a robust login experience.
- **Layout:** Built with a mix of Radix-based components and custom brutalist styling.

### 3. The Inventory (Product Grid)
- **Dynamic Search:** Real-time filtering by name or intensity tag.
- **Industrial UI:** High-contrast cards with 8px hard shadows and grayscale-to-color hover transitions.
- **Quick-Action System:** Add items to the cart directly from the inventory view with instant feedback.

### 4. User Authentication
- **Secure Schema:** Login validation powered by Zod.
- **Tester Access Tab:** A dedicated testing section allowing developers/reviewers to auto-fill credentials for `muneeb` or `faraz`.

## 🎨 DESIGN PHILOSOPHY (NEO-BRUTALISM)

- **Palette:** `#FFD700` (Cyber Yellow), `#000000` (Deep Black), `#FFFFFF` (Pure White).
- **Borders:** Consistent `4px` and `8px` solid black strokes.
- **Shadows:** Hard-edged, non-blurred offsets for a "printed" look.
- **Typography:** Heavy-weight, italicized, and uppercase font-sets to mimic industrial warning labels.
